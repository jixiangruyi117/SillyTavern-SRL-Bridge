# 更新记录

## 0.3.8 - 2026-07-23

- 安卓同浏览器连接不再经过页面扩展中间页和 iframe，改为从酒馆直接打开 SRL 顶层页面，避免移动浏览器丢失 `window.opener`。
- 设备码会话只信任酒馆创建会话时锁定的 SRL 地址；加入端只提交短时设备码，不再因重定向或不同入口误报“SRL 来源与设备码不一致”。

## 0.3.7 - 2026-07-23

- 设备码加入入口升级为 `/join-v2`，避免新版资源库误连旧服务端插件后继续打开 iframe 空白资源库。
- 旧服务端插件会明确连接失败并提示升级，不再静默回退到会产生独立 IndexedDB 的旧流程。

## 0.3.6 - 2026-07-23

- 跨浏览器设备码不再把 SRL 嵌入酒馆来源的 iframe；原 HTTPS 资源库保持顶层页面，中继窗口只负责转交通信端口，避免浏览器存储分区生成一份看似空白的新资源库。
- 中继页改为轻量状态窗口，并明确提示保留窗口和返回原资源库；加入页覆盖酒馆的 COOP 响应头以保留受控 opener 通道。

## 0.3.5 - 2026-07-23

- Termux/Linux/macOS 默认安装不再下载 GitHub Release 附件，直接获取服务端插件的两个源码文件；Raw GitHub 失败时自动尝试 jsDelivr 镜像。
- 离线 `--package` 仍支持 ZIP；`unzip` 仅因 Windows 路径分隔符返回警告码时继续检查实际解压结果，不再提前退出。
- 发布脚本改为显式写入 ZIP 标准 `/` 路径，避免不同 Windows PowerShell/压缩工具产生反斜杠条目。

## 0.3.4 - 2026-07-23

- 修复跨浏览器设备码被酒馆登录身份二次拦截、导致另一浏览器无法加入的问题；设备码仍为两分钟一次性凭证，并继续使用随机会话令牌和六位确认码。
- 同浏览器连接改用每次独立的中继窗口，断开时关闭旧窗口，避免浏览器复用残留页面后出现“没有找到酒馆窗口”。
- 中继页在需要登录 SRL 时显示明确进度，登录完成后继续自动配对。

## 0.3.3 - 2026-07-23

- Windows 安装器取消必填固定路径：自动搜索常见目录，多个安装可交互选择，并兼容全局模式的独立 `config.yaml`。
- 新增 Android/Termux、Linux 与 macOS 安装器，自动识别官方常见的 `~/SillyTavern`，也允许显式指定任意路径。
- 完整发布包内置两种安装脚本；文档明确 iPhone/iPad 只需在真正运行酒馆的电脑或服务器安装。

## 0.3.2 - 2026-07-23

- 增加 Windows 服务端插件一键安装脚本，自动下载最新 Release、备份旧版并启用 `enableServerPlugins`。
- 安装脚本显式使用 TLS 1.2，兼容旧版 Windows PowerShell 访问 GitHub。
- 自动下载改用固定的 latest Release 资源，不依赖匿名 GitHub API 配额。
- Git 安装文档统一使用仓库 `.git` 地址，并增加重复目录、重复扩展和网络失败排查。
- 移除扩展清单中已弃用的 `requires` / `optional` 空字段。

## 0.3.1 - 2026-07-23

- 发布公开 GitHub 安装源，可在 SillyTavern 中直接粘贴仓库地址安装和更新页面扩展。
- 补充扩展主页、功能说明、兼容版本和服务端插件独立安装教程。
- 服务端中继继续通过 Release ZIP 与完整安装包分发，不让页面扩展越权写入服务端目录。

## 0.3.0 - 2026-07-23

- 新增短期设备码服务端中继，支持不同浏览器及移动端配对。
- 正则拆分为全局、角色卡、预设三种作用域，导入时写回正确所属对象。
- 分块传输增加逐块确认，限制内存队列，避免大文件一次性堆积。

## 0.2.0 - 2026-07-23

- 新增全局正则、快速回复组和酒馆主题双向互传。
- 在扩展中显示当前浏览器名称，明确同浏览器直连的数据边界。
- 补充跨浏览器设备码中继的安全与能力说明。

## 0.1.1 - 2026-07-23

- 酒馆设置改为默认折叠的原生扩展抽屉，重做连接状态、地址提示、操作区和窄栏响应式排版。
- 手机从局域网地址打开酒馆时，默认 SRL 地址自动使用相同主机的 5173 端口，不再误用手机自身的 127.0.0.1。

## 0.1.0 - 2026-07-23

- 首次提供 SRL 与 SillyTavern 的角色卡、世界书和预设双向互传。
- 增加一次性配对、同源中继、分块传输、完整性校验和冲突策略。
- 增加桌面与移动端适配的 SRL 资源选择界面。
## 0.3.8 - 2026-07-25

- 清理仓库内旧版 release ZIP，只保留当前 `0.3.8` 和 latest 服务端插件包，减少 SillyTavern 从 GitHub 安装/更新时的仓库体积。
## 0.3.8 - 2026-07-25

- 调整服务端插件安装器：新版本安装成功后默认删除旧插件临时备份；安装失败时自动恢复旧插件；需要保留旧版时可追加 `--keep-backup` / `-KeepBackup`。
